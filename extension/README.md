# Mineradio Connector

Mineradio Connector 是给 Mineradio Web 使用的 Chrome / Edge Manifest V3 扩展。它负责在用户本机读取浏览器里的音乐网页登录态，并向 Mineradio Web 提供搜索、歌单、歌词和播放地址探测能力。

扩展不会把 Cookie 原文发送给 Mineradio Web 页面；会话信息保存在浏览器扩展环境中。

当前版本：`0.5.6`

## 安装

1. 下载 `mineradio-connector.zip`。
2. 解压到固定文件夹。
3. Edge 打开 `edge://extensions`，Chrome 打开 `chrome://extensions`。
4. 开启“开发人员模式”。
5. 点击“加载解压缩的扩展”，选择解压后的文件夹。
6. 打开或刷新 Mineradio Web。

更新扩展文件后，需要在扩展管理页点击“重新加载”。

## 使用建议

- 先在对应音乐网页版登录，再回到 Mineradio Web 搜索或导入歌单。
- 如果已经登录但仍显示受限，可以在对应音乐网页内播放一首歌后，回到扩展面板点击重试。
- QQ 音乐和酷狗的播放授权更依赖网页端即时状态，Connector 会优先使用已缓存会话，但平台接口仍可能临时拒绝播放地址。
- VIP 信息只有在平台接口能明确确认时才显示；无法确认时不会强行显示会员状态。

## 平台能力

| 平台 | 当前能力 |
| --- | --- |
| 网易云音乐 | 登录态保存、账号状态、首页推荐、我的歌单、搜索、歌词、播放探测、外链歌单导入 |
| QQ 音乐 | 登录态保存、账号状态、我的歌单、搜索、播放探测、外链歌单导入 |
| 酷狗音乐 | 登录态检测、搜索、播放探测、分享歌单导入；账号“我的歌单”暂不伪造支持 |

酷狗播放状态采用严格策略：只有真实播放地址探测成功，才显示为可播放。登录态和 VIP 状态可以被记忆，但不会被当成播放成功。

## 网络与代理

扩展请求会跟随浏览器或系统网络环境。如果你启用了代理，建议把以下域名设为国内直连：

- `music.163.com`
- `*.music.126.net`
- `y.qq.com`
- `u.y.qq.com`
- `c.y.qq.com`
- `*.qqmusic.qq.com`
- `*.kugou.com`

海外出口可能触发地区限制或风控，导致网页登录可见但播放地址不可用。
