Mineradio Bridge

Bridge 是运行在你电脑上的本地服务，网页播放器通过它访问在线音源、歌词、封面和音频代理。它不会直接读取你本机已安装的网易云音乐或 QQ 音乐客户端登录态。

普通用户

下载 Windows Bridge 压缩包，解压后双击 Start-Mineradio-Bridge.cmd。启动成功后会自动打开配对页面。

下载 Bridge 构建 已启动，立即配对 返回播放器

首次版本先提供 zip 包。解压后的 Install-Mineradio-Bridge-Startup.cmd 可设置开机自启。

开发测试

如果你是源码测试，也可以在项目目录双击 Start-Mineradio-Bridge.cmd，或执行：

npm install
npm run bridge

平台登录

Bridge 配对只代表网页已连接本机代理。网易云音乐和 QQ 音乐账号仍需分别登录。Web 版优先使用 Cookie 导入，避免网易云旧二维码接口触发 App 风控警告。

为什么不能自动启动

浏览器网页不能自动启动本机程序，这是系统和浏览器的安全限制。下载安装 Bridge 后，可以通过开机自启脚本让它随 Windows 登录自动运行。

为什么不能直接用 App 登录态

桌面 App 的登录态在各自私有存储里，通常加密并和客户端绑定。Bridge 不会读取这些私有数据；它只保存你主动导入或通过受控登录流程获得的网页会话。
